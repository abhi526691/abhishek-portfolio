Abhishek Pandey's portfolio — a story-driven homepage built with Next.js (App Router).

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `src/components/Portfolio.tsx` — the whole homepage: hero, scroll-driven story chapters, the six-discipline work catalogue, and contact section.
- `src/lib/fonts.ts` — Instrument Serif, Inter, and JetBrains Mono via `next/font/google`.

## Deploy

Deploy on [Vercel](https://vercel.com/new).
